# My-hexlet-js Project

Мой первый проект на Hexlet

Проект для изучения JavaScript

## Установка зависимостей

Для установки всех необходимых зависимостей проекта используйте команду

```bash
make install
```

## Линтинг

Для проверки кода на соответствие стандартам используйте команду:

```bash
make lint
```
Для автоматическогоисправления ошибок используйте команду:

```bash
make lint-fix
```
